== Description ==

The coming soon plugin used with ease

It hides your site when you are building it, as long as you are logged in as admin, you would be the only one able to see the site.


###  Features:

* Works with any WordPress Theme
* Free Coming soon page templates (light and dark themed)
* Easy to activate

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3.Go to the settings page and activate it . Add your chosen heading and sub heading as well as an image if you wish.
